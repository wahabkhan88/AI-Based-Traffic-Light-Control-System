# 🚦 AI Based Traffic Light Control System

## 📌 Project Overview

AI Based Traffic Light Control System is an Artificial Intelligence project that simulates an intelligent traffic management system.

The system uses Reinforcement Learning (Q-Learning) to dynamically control traffic signals based on:
- Traffic density
- Waiting time
- Emergency vehicle priority

The goal is to optimize traffic flow and reduce congestion by automatically selecting the lane that requires the green signal.

## ✨ Features

- 🚗 Simulates 4 traffic lanes
- 🚦 AI controlled traffic signals
- 📊 Detects traffic density
- ⏳ Tracks vehicle waiting time
- 🚑 Emergency vehicle detection
- 🧠 Learns optimal signal decisions using Q-Table
- 📈 Displays traffic performance graphs

---

## ⚙️ Algorithm

1. Initialize traffic lanes and Q-table
2. Generate random traffic density
3. Detect emergency vehicles
4. Create current traffic state
5. Select lane using AI decision making
6. Give priority based on:
   - Vehicle density
   - Waiting time
   - Emergency vehicles
7. Update traffic flow
8. Calculate reward
9. Train Q-learning model
10. Display performance results
